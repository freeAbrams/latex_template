# LaTeX Template

This folder is the baseline template for new paper workspaces. Start a new paper by copying or forking this directory, then open the copied directory itself in VS Code.

## Files

- `main.tex`: template entry file for new papers
- `paper_style.sty`: local style definition
- `build.sh`: local build script using `../.tools/tectonic`
- `.vscode/settings.json`: workspace-local LaTeX Workshop settings
- `pic/example-image.png`: sample figure asset
- `AGENTS.md`: local workflow and naming guidance for Codex sessions

## Shared Resources

- Fonts are read from `../.fonts/`
- Tectonic is stored in `../.tools/`
